# AppleMapsUnlock

This project contains a sample iOS application. The project uses Swift and requires Xcode 15 or later.

## Setup

1. Install **Xcode 15** or newer from the Mac App Store or the Apple developer website.
2. Clone this repository and open `AppleParkUnlock.xcodeproj` in Xcode.
3. Select an iOS Simulator (e.g. *iPhone 15*) and build/run the project with **Cmd+R**.

## Running Tests

You can run the unit and UI tests from Xcode using **Cmd+U**. Alternatively, you can run them from the command line with `xcodebuild`:

```bash
xcodebuild test -scheme AppleParkUnlock -destination 'platform=iOS Simulator,name=iPhone 15,OS=latest'
```
